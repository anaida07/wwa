# Application Template
**React** + **Express** + **TypeScript** Application Template

## Setup Development Machine
1. Clone the repo `git clone git@github.com:anaida07/wwa.git`

2. `cd wwa`

3. Open client
```
cd client
npm run dev
```

4. Open server
```
cd server
npm start
```

5. Open `http://localhost:8080` in browser
